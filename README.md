# Triple Espresso

Landing page desarrollada para la cafetería Triple Espresso, con formulario para la reservación directa de mesas y redireccionamiento a redes sociales.
Desarrollada en HTML5 + CSS.
Se aplicará JavaScript para el envío de información del formulario y la conexión a base de datos no relacional por medio de MongoDB. Adicionalmente, se podrá implementar una agenda a tiempo real con los horarios disponibles para la reservación de mesas.
